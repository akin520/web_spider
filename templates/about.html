使用说明

小小爬虫是基于urllib urllib2 BeautifulSoup celery redis mysql的网页爬虫工具

安装模块

pip install celery redis

pip install requests

pip install BeautifulSoup4

pip install flower

添加网页

URL：是要抓取的页面，如图中的http://news.163.com

过滤路径：抓取包含有这么路径的地址，如/ooxx，要抓取的链接中必须有/ooxx这个路径，如http://news.163.com/aa/ooxx/1.html,http://news.163.com/ooxx/index.html都为合法

下载名：打包后的名字

xpath：如a，src等

xlink：如a中属性href,和xpath对应，属性只能对应一个

运行

运行celery: celery -A spider worker -c 10 --loglevel=INFO

运行celery监控工具: celery -A spider flower --port=5555

运行web.py: python web_service.py
